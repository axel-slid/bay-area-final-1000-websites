# Bay Area Final 1000 Websites

Static bundle containing 1000 generated local-business websites.

- Production host: `https://bay-area-final-1000-websites.vercel.app/`
- Source CSV in this repo: `deliverables/bay-area-final-1000-websites.csv`
- Driving report in this repo: `deliverables/bay-area-final-1000-driving-report.md`
- Confirmed owned-site removals: `deliverables/confirmed-owned-site-removals.csv`

The deployed build publishes only the active 1000 business slugs. Removed leads return 404.
